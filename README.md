# 💉 Blood Bank Management System

A modern web application designed to manage and streamline the operations of blood banks. This system helps hospitals and organizations efficiently handle blood donor records, inventory tracking, and blood request management — ensuring that life-saving blood is always available when needed.

---

## 🌟 Key Features

- 👤 **Donor & Patient Management**  
  Add, view, and manage donor and patient records with essential details.

- 🩸 **Real-time Blood Inventory**  
  Keep track of available blood units for each blood group.

- 🔍 **Smart Blood Search**  
  Quickly find donors or blood availability based on blood type.

- 📥 **Blood Request Handling**  
  Allow hospitals or patients to request blood and track its status.
